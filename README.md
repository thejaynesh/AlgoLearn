# AlgoLearn

An interactive educational platform for visualizing algorithms.

## Features

- **Sorting Algorithms**: Visualize Merge Sort, Bubble Sort, Selection Sort, Insertion Sort, and Quick Sort.
- **Pathfinding**: Visualize Dijkstra's algorithm on an interactive grid.
- **Responsive Design**: Built with React and Tailwind CSS for a modern, responsive UI.

## Tech Stack

- **Frontend**: React, Tailwind CSS, Vite
- **Deployment**: GitHub Pages (via GitHub Actions)

## Getting Started

1.  Clone the repository:
    ```bash
    git clone https://github.com/yourusername/AlgoLearn.git
    ```
3.  Install dependencies:
    ```bash
    npm install
    ```
4.  Run the development server:
    ```bash
    npm run dev
    ```

## Deployment

This project is configured to automatically deploy to GitHub Pages when changes are pushed to the `main` branch.

1.  Push changes to `main`.
2.  Go to your repository Settings > Pages.
3.  Set the source to **GitHub Actions**.
